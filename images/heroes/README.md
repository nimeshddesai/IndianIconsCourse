# Hero Images Directory

This directory contains portrait photographs of the military heroes featured in the course.

## Image Requirements

- **Format**: JPG or PNG
- **Dimensions**: Recommended 400×500 px (portrait orientation)
- **Naming**: Lowercase, hyphen-separated — e.g. `vikram-batra.jpg`
- **Size**: Keep under 200 KB for fast loading
